🚀 Cloud-Hosted Interactive Portfolio Website
A modern, serverless, and highly responsive personal portfolio website designed for a Cloud Engineer & Full-Stack Developer. Built with Tailwind CSS, enhanced with AOS (Animate On Scroll), and architected to be deployed on AWS Cloud using an automated CI/CD Pipeline.

🌟 Key Features
Glassmorphic UI & Dark Theme: Styled using Tailwind CSS with glassmorphism effects and modern design aesthetics.

Scroll Animations: Integrated with AOS library for smooth fade-in, zoom, and slide transitions on scroll.

Fully Responsive: Optimized across mobile, tablet, and desktop views.

Cloud Architecture Ready: Designed to be hosted on Amazon S3 and distributed worldwide via Amazon CloudFront (CDN) with SSL/TLS security.

Automated CI/CD: Configured with AWS CodePipeline to automatically fetch updates from GitHub on every push and deploy live within seconds.

🛠️ Tech Stack & Services Used
Frontend: HTML5, Tailwind CSS, JavaScript (ES6+)

Libraries: AOS (Animate On Scroll), FontAwesome Icons, Google Fonts

Cloud Services (AWS):

Amazon S3: Static Website Hosting & Asset Storage

Amazon CloudFront: Content Delivery Network (CDN) with HTTPS

AWS CodePipeline: Continuous Integration & Continuous Deployment (CI/CD)

Version Control: Git & GitHub

☁️ AWS Deployment Architecture
Developer Push ---> GitHub Repo ---> AWS CodePipeline ---> AWS S3 Bucket ---> AWS CloudFront (CDN) ---> End User

Code is pushed to the main branch of this GitHub repository.

AWS CodePipeline detects the commit and triggers a deployment stage automatically.

Updated static assets are synchronized to the Amazon S3 bucket.

Amazon CloudFront caches and delivers the updated assets globally with low latency.
